# Contributor Covenant Code of Conduct

## Our Pledge

We as members, contributors, and leaders pledge to make participation in our
community a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:
- Demonstrating empathy and kindness.
- Being respectful of differing opinions, viewpoints, and experiences.
- Giving and accepting constructive feedback gracefully.
- Prioritizing community and project well-being over individual ego.

Examples of unacceptable behavior include:
- Harassment, discrimination, or hateful conduct.
- Trolling, insulting or derogatory comments, and personal attacks.
- Publishing private information without explicit permission.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of
conduct and may remove, edit, or reject contributions and comments that are not
aligned with this Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces and in public spaces when
an individual is representing the project.

## Enforcement

To report unacceptable behavior, open a private security report if available or
contact project maintainers through repository communication channels.
